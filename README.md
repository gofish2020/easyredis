## 使用说明

### 单机版
- 使用`./test.sh`命令启动单机版服务端
- 使用`./redis-cli.sh`命令启动官方端redis客户端，连接服务（需要你本机自己安装redis-cli并加入到环境变量中）

- 使用`./redis-cluster0.sh` `./redis-cluster1.sh` `./redis-cluster2.sh`命令启动3个服务端
